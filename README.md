b2tool
======

## bbtool is a command line tool to manage BitBucket

### List of commands and options

* auth
    * login
        * --username
        * --password

* pullrequest

    * listall
    * oldest
    * accept

## How to use

<pre><code>$ b2tool command options</pre></code>

** For example

<pre><code>$ b2tool pullrequest listall --owner jesuejunior --repo projectA </pre></code>
